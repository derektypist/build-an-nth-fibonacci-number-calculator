# Build an Nth Fibonacci Number Calculator

Build an Nth Fibonacci Number Calculator using Dynamic Programming.

## Technologies

Uses Python3 and Dynamic Programming.

## Deployment

Copy `main.py` into the text editor of your choice.  If using terminal, use the command `python3 main.py` to run it.
